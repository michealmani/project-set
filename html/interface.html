Interface:

  • Interfaces can have abstract methods and variables. It cannot have a method body. It is used to achieve abstraction and multiple inheritance in Java.
  • An interface is a group of related methods with empty bodies
syntax:

interface {
//methods
}

To access the interface methods, the interface must be "implemented" by another class with the implements keyword . The body of the interface method is provided by the "implement" class.
In other words, Interface fields are public, static and final by default, and the methods are public and abstract.

The relationship between classes and interfaces:
Example interface :
interface printable {
void print();
void show();
}
class inter implements printable {
public void print() {
System.out.println("Hello I am print"); }
public void show() {
System.out.println("Hello I show that content");
}
public static void main(String []args) {
inter obj=new inter();
obj.print();
obj.show();
}
}
Multiple inheritance Example:
interface Printable{
void print();
}
interface Showable{
void show();
}
class test implements Printable,Showable{
public void print(){System.out.println("Hello");
}
public void show(){System.out.println("Welcome");
}
public static void main(String args[]){
test obj = new test();
obj.print();
obj.show();
}
}

Interface inheritance:

A class implements an interface, but one interface extends another interface.

interface Printable{
void print();
}
interface Showable extends Printable{
void show();
}
class inter implements Showable{
public void print(){System.out.println("Hello");
}
public void show(){System.out.println("Welcome");
}
public static void main(String args[]){
inter obj = new inter();
obj.print();
obj.show();
}
}
