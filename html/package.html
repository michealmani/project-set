Java Package:

A java package is a group of similar types of classes, interfaces and sub-packages.

Two forms

built-in package - java, lang, awt, io, util, sql etc.
o java.lang − bundles the fundamental classes
o java.io − classes for input , output functions are bundled in this package

o user-defined package

Simple example of java package:

The package keyword is used to create a package in java.

//save as Simple.java
package mypack;
public class Simple{
public static void main(String args[]){
System.out.println("Welcome to package");
}
}

Class_1.java

package packages;
import java.io.*;
public class class_1 {
int basic;
double da,ta,allowance,pf,it,dect,netpay;
public class_1(int a) {
basic=a;
da=basic*0.05;
ta=basic*0.08;
allowance=da+ta;
pf=basic*0.09;
it=basic*0.03;
dect=pf+it;
netpay=(basic+allowance)-dect;
}
public void display() {
System.out.println("\n Basic Salary"+basic);
System.out.println("\n Dearness Allowance"+da);
System.out.println("\n Travelling Allowance"+ta);
System.out.println("\n Provident Fund"+pf);
System.out.println("\n Income Tax"+it);
System.out.println("\n Deduction"+dect);
System.out.println("\n Net Salary"+netpay);
}
}
import packages.class_1;
import java.io.*;
class main_class
{
public static void main(String args[]) {
class_1 ab=new class_1(50000);
ab.display();
}
}
